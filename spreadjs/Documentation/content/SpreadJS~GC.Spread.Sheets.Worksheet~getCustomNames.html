    getCustomNames Method
    In This Topic
    Gets all custom name information.
    Syntax
    • Javascript (Usage)
    • Javascript (Specification)
    var instance = new GC.Spread.Sheets.Worksheet(name);
var value; // Type: any
value = instance.getCustomNames();
    function getCustomNames() : any;

    Return Value

    The type GC.Spread.Sheets.NameInfo stored in an array.
    Example
    This example gets the custom names.
    • Javascript
    activeSheet.setValue(0, 0, "customName1");
activeSheet.setValue(0, 1, "customName2");
activeSheet.addCustomName("customName1", "=12", 0, 0);
activeSheet.addCustomName("customName2", "Average(20,45)", 0, 0);
activeSheet.setFormula(1, 0, "customName1");
activeSheet.setFormula(1, 1, "customName2");
