    Background Image
    In This Topic

    Worksheets with huge amount of data and no images often look dull and boring.

    While working with worksheets in SpreadJS, you can insert any picture of your choice as a background image. This can be helpful especially when you're working for an organization where you don't just want to create plain spreadsheets but also leave an impact of your brand value and reputation via embedding multimedia to the worksheets.

    If you want to display a distinct background for various spreadsheet elements, you can add different background images to the cells, columns, rows, or widgets, as per your requirements and preferences. You can also add a background image to a cell, column, row, widget or Spread.

    The following operations can be done while inserting background images in SpreadJS:

    1. Users can add a background image to one or multiple worksheets, cells, rows, columns and widgets.
    2. Users can remove an existing background image from the worksheet, cells, rows, columns and widgets.
    3. Users can display watermark information on every page via adding an image that includes the watermark logo or other information in header/footer area.
    4. Users can resize the embedded images.
    5. Users can modify the layout of the background image for cells, columns and rows in the worksheet. This can be done by specifying whether the image should stretch to fill the area, should be centered or zoomed while keeping the original aspect ratio, or the original size is to be maintained.

    An example screenshot with background images is shown below.

    You can load local base 64 bitmap images or images from an image Uri.

    You can specify the background image layout for the cell, column, row, or Spread. You can use the ImageLayout enumeration to specify whether the image is stretched to fill the area, centered, zoomed with the original aspect ratio, or keeps the original size.

    Using Code

    This example loads images in a cell, column, and row.

    JavaScript
    Copy Code 
    var cellImage = document.getElementById("circle").src;
var rowImage = "./css/images/quarter1.png";
var colImage = document.getElementById("logo").src;
sheet.getCell(3, 3).backgroundImage(cellImage);
sheet.getRange(1, -1, 1, -1).backgroundImage(rowImage);
sheet.getRange(-1, 1, -1, 1).backgroundImage(colImage);

    This example centers the image in the row.

    JavaScript
    Copy Code 
    var cellImage = document.getElementById("circle").src;
var rowImage = "./css/images/quarter1.png";
var colImage = document.getElementById("logo").src;
activeSheet.getCell(3, 3).backgroundImage(cellImage);
activeSheet.getRange(1, -1, 1, -1).backgroundImage(rowImage);
activeSheet.getRange(-1, 1, -1, 1).backgroundImage(colImage);
activeSheet.getRange(1, -1, 1, -1).backgroundImageLayout(GC.Spread.Sheets.ImageLayout.Center);
