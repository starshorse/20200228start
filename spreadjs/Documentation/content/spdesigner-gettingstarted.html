    Installation

    You can install the SpreadJS Designer by downloading the SpreadJS ZIP file (SpreadJS.Release.xxxx.zip) to a temporary directory on your system and then execute the preferred setup file after unzipping the files to a directory.

    Note: The SpreadJS Designer is already included in the SpreadJS product download.
    Install Directory File Description
    \SpreadJS.Release.xxxx\Designer\Designer Runtime Main directory for product
    SpreadJS-Designer.xxx.exe Setup for Windows
    SpreadJS-Designer.xxx.dmg Setup for MacOS
    SpreadJS-Designer.xxx.AppImage Setup for Linux

    License Information

    SpreadJS Designer supports the following types of licenses:

    Unlicensed

    The unlicensed version allows you to use all features of SpreadJS Designer for 30 days.

    The top bar in the application window displays the following message:

    "Evaluation version (30 day(s) left). Select the LOCK button to activate the Designer before the trial ends."

    A watermark is displayed at the bottom right corner:

     

     

    After the expiration of 30 days trial period, all the import/export actions are disabled. When the Designer is opened, the following message appears at the top:

    "License not found. Select the LOCK button to activate the Designer." 

    And the license dialog is shown as below:

     

    Evaluation License

    The evaluation license allows you to use all features of SpreadJS Designer.

    However, the top bar in the application window displays the following message:

    "Evaluation version (30 day(s) left). Select the LOCK button to activate the Designer before the trial ends."

     A watermark is displayed at the bottom right corner.

     

    Licensed

    The licensed version allows you to use all features of SpreadJS Designer.

    Apply License

    The license key of SpreadJS Designer should be set in the "License Information" dialog.
