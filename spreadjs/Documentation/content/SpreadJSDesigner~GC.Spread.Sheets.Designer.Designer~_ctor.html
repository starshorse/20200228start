    Designer Constructor
    In This Topic
    Represent a Designer with the specified hosted DOM element, custom config and an existing spread.
    Syntax
    • Javascript (Usage)
    • Javascript (Specification)
    var instance = new GC.Spread.Sheets.Designer.Designer(host, config, spread, spreadOptions);
    function Designer( 
   host : undefined,
   config : Object,
   spread : Object,
   spreadOptions : Object
) : Designer;

    Parameters

    host
    This is the HTML area that the Designer Component mounts.
    config
    The designer config object.
    spread
    The workbook instance.
    spreadOptions
    The workbook initialization options.
