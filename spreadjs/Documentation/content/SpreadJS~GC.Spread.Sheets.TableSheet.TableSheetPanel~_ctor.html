    TableSheetPanel Constructor
    In This Topic
    Represents a table sheet panel with the specified name, table sheet, host element and options setting.
    Syntax
    • Javascript (Usage)
    • Javascript (Specification)
    var instance = new GC.Spread.Sheets.TableSheet.TableSheetPanel(name, tableSheet, host, options, options.showSource);
    function TableSheetPanel( 
   name : string,
   tableSheet : TableSheet,
   host : HTMLElement,
   (optional) options : Object,
   (optional) options.showSource : ShowSourceOptions
) : TableSheetPanel;

    options
    The initialization options.
    options.showSource
    How to show a column value.
