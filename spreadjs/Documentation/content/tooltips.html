    Tooltips
    In This Topic

    SpreadJS provides the option to display tooltips when resizing or scrolling in the control or when dragging or moving the data.

    Tooltip for Drag-Drop Action

    You can choose whether to display the tooltip when performing drag-drop action by setting the showDragDropTip property of the Workbook class. The property accepts a boolean value and its default value is True. The tooltip displays the cell value during drag-drop operation in either A1 or R1C1 style depending on the setting of the referenceStyle property in the Workbook class.

    The following image displays the A1 reference style for the drag-drop tip.

    JavaScript
    Copy Code 
    spread.options.showDragDropTip = true; // Set False to hide.

    Tooltip for Drag-Fill Action

    You can choose whether to display the tooltip for the drag-fill action when automatically filling range of cells in a sequential or an evenly spaced interval by setting the showDragFillTip property of Workbook class. This property accepts boolean value and its default value is True.

    The following image displays a drag fill tip.

    JavaScript
    Copy Code 
    spread.options.showDragFillTip = true; // Set False to hide.

    Tooltip for Scrollbar action

    You can choose whether to display the tooltip while scrolling the horizontal or vertical bar to show the row or column name by setting the showScrollTip property of Workbook class.
    This property accepts ShowScrollTip enumeration values as none (default), horizontal, vertical or both.

    The following image displays a scroll tip on horizontal bar.

    JavaScript
    Copy Code 
    spread.options.showScrollTip = GC.Spread.Sheets.ShowScrollTip.Both;
// Or
// spread.options.showScrollTip = 3;

    Tooltip for Resize action

    You can choose whether to display the tooltip when resizing rows or columns to show the pixel size by setting the showResizeTip property of Workbook class.
    This property accepts ShowResizeTip enumeration values as none (default), column, row or both.

    The following image displays a resize tip.

    JavaScript
    Copy Code 
    spread.options.showResizeTip = GC.Spread.Sheets.ShowResizeTip.Both;
// Or
// spread.options.showResizeTip = 3;
