    Grouping API Changes
    In This Topic

    The following changes have been made to grouping:

    V9 V13
    GcSpread.Sheets.RangeGroup GC.Spread.Sheets.Outlines.Outline
    GcSpread.Sheets.GroupState GC.Spread.Sheets.Outlines.OutlineState
    GcSpread.Sheets.RangeGroupInfo GC.Spread.Sheets.Outlines.OutlineInfo
    GcSpread.Sheets.RangeGroupDirection GC.Spread.Sheets.Outlines.OutlineDirection
    GcSpread.Sheets.UndoRedo.ColumnGroupUndoAction GC.Spread.Sheets.Commands.outlineColumn
    GcSpread.Sheets.UndoRedo.ColumnUngroupUndoAction GC.Spread.Sheets.Commands.removeColumnOutline
    GcSpread.Sheets.UndoRedo.ColumnGroupExpandUndoAction GC.Spread.Sheets.Commands.expandColumnOutline
    GcSpread.Sheets.UndoRedo.ColumnGroupHeaderExpandUndoAction GC.Spread.Sheets.Commands.expandColumnOutlineForLevel
    GcSpread.Sheets.UndoRedo.RowGroupUndoAction GC.Spread.Sheets.Commands.outlineRow
    GcSpread.Sheets.UndoRedo.RowUngroupUndoAction GC.Spread.Sheets.Commands.removeRowOutline
    GcSpread.Sheets.UndoRedo.RowGroupExpandUndoAction GC.Spread.Sheets.Commands.expandRowOutline
    GcSpread.Sheets.UndoRedo.RowGroupHeaderExpandUndoAction GC.Spread.Sheets.Commands.expandRowOutlineForLevel

    The following table lists API changes:

    V9 V13 Namespace
    setDirection(direction)/getDirection() direction(direction) GC.Spread.Sheets.Outlines.Outline
    isGroupEnd(index, indexNext, processLevel) isGroupEnd(index, processLevel) GC.Spread.Sheets.Outlines.Outline
    setState(value)/getState() state(value) GC.Spread.Sheets.Outlines.OutlineInfo

    • GcSpread.Sheets.UndoRedo.GroupExtent
