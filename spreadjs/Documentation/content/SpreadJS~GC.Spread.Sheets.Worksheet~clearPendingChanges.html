    clearPendingChanges Method
    In This Topic
    Clears the dirty, insert, and delete status from the current worksheet.
    Syntax
    • Javascript (Usage)
    • Javascript (Specification)
    var instance = new GC.Spread.Sheets.Worksheet(name);
var value; // Type: any
value = instance.clearPendingChanges(clearChangeInfo,
                                     clearChangeInfo.row,
                                     clearChangeInfo.col,
                                     clearChangeInfo.rowCount,
                                     clearChangeInfo.colCount,
                                     clearChangeInfo.clearType);
