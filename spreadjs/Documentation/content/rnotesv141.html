    Enhancements from the Previous Release

    The following features and enhancements have been incorporated in this version of the product.

    • Users can now set the position of axis data labels in charts.
    • Arrange shapes using alignment, distribution, and snap options - Users can align multiple shapes, distribute them, and snap them to other shapes or grids.
    • TrailingRow without having blank space after data rows - Users can specify whether to have frozen trailing rows or columns or both in a sheet.
    • Set a limit on the number of undo and redo operations - Users can get the undo and redo stack as well as set a limit on number of undo and redo operations.
    • Change tab strip position and width - Users can set the position of tab strip relative to the workbook and change the width of tab strip.
    • Paste or skip data in invisible range - Users can now control whether to paste the data in an invisible range or not.
    • Support to import Excel file with customizable date format - Users can now customize the date format based on system locale settings while importing an Excel file.
    • SpreadJS supports WEBSERVICE function which helps in fetching data from a web service on the Internet or Intranet.
    • SpreadJS supports FILTERJSON function which helps in parsing data from a JSON string.
    • IMAGE sparkline - Users can now place an image in a cell using IMAGE sparkline function.
    • SpreadJS now supports enhancing Check Box list, Radio Button list, and Check Box cell width size.
    • Users can set the orientation of chart axis.
    • Users can now change the hover backColor of cell button when it is visible or enabled.
    • Users can create histogram sparkline to represent the frequency distribution of a dataset.
    • Users can create gauge KPI sparkline to visualize the performance of metrics with respect to KPI values.
    • Users can now add Pivot Tables and perform various pivot related operations in SpreadJS.
    • Users can now use fill effects such as solid fill, gradient fill, picture fill, and texture fill in shapes.
    • Hyperlinks on Shapes - Users can add hyperlinks on shapes, connector shapes, or group shapes.
    • Camera Shape - Users can add dynamic images of a referenced area in a spreadsheet.
    • Users can add and edit text in shapes.
    • Users can now add different types of compound lines in shapes to customize their appearance.

    Breaking Changes from the Previous Release

    The following breaking change has been introduced prior the last release of the product.

    • The name argument of LET function should not contain a '.'. All the pre-existing Excel and ssjson files using LET function and containing '.' in the variable names will be silently upgraded to valid variable names. SpreadJS will convert the '.' to '_', to match the updated Excel behavior.
