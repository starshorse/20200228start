    Sparkline Functions
    In This Topic

    Spread provides the following sparklines functions, listed alphabetically.

    Function Description
    AREASPARKLINE This function returns a data set used for representing an area sparkline.
    BOXPLOTSPARKLINE This function returns a data set used for representing a boxplot sparkline.
    BULLETSPARKLINE This function returns a data set used for representing a bullet sparkline.
    CASCADESPARKLINE This function returns a data set used for representing a cascade sparkline.
    COLUMNSPARKLINE This function returns a data set used for representing a column sparkline.
    GAUGEKPISPARKLINE This function returns a data set used for representing a gauge KPI sparkline.
    HBARSPARKLINE This function returns a data set used for representing an Hbar sparkline.
    HISTOGRAMSPARKLINE This function returns a data set used for representing a histogram sparkline.
    IMAGE This function input an URL or a base64 string to show an image in cell.
    LINESPARKLINE This function returns a data set used for representing a line sparkline.
    MONTHSPARKLINE This function returns a data set used for representing a month sparkline.
    PARETOSPARKLINE This function returns a data set used for representing a pareto sparkline.
    PIESPARKLINE This function returns a data set used for representing a pie sparkline.
    RANGEBLOCKSPARKLINE This function returns a rangetemplate.
    SCATTERSPARKLINE This function returns a data set used for representing a scatter sparkline.
    SPREADSPARKLINE This function returns a data set used for representing a spread sparkline.
    STACKEDSPARKLINE This function returns a data set used for representing a stacked sparkline.
    VARISPARKLINE This function returns a data set used for representing a variance sparkline.
    VBARSPARKLINE This function returns a data set used for representing a Vbar sparkline.
    WINLOSSSPARKLINE This function returns a data set used for representing a win/loss sparkline.
    YEARSPARKLINE This function returns a data set used for representing a year sparkline.
