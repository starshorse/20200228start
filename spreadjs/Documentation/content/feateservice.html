    Excel Import and Export
    In This Topic

    You can use the Excel Import and Export component to load and export Excel-formatted files. You can also load or save Excel-formatted files on the client-side.

    For more information, refer to Import and Export Excel Files on Client-Side.

    Note: The Excel Import and Export Server Component is not supported from SpreadJS version 11.1.0 onwards. If you want to use this functionality, please switch to GrapeCity Documents for Excel.
