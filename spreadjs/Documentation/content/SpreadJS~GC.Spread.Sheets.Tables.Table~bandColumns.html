Whether to display an alternating column style.

    bandColumns Method
    In This Topic
    Gets or sets a value that indicates whether to display an alternating column style.
    Syntax
    • Javascript (Usage)
    • Javascript (Specification)
    var instance = new GC.Spread.Sheets.Tables.Table(name,
                                                row,

                                                style,
                                                options,
                                                options.showHeader,
                                                options.showFooter,
                                                options.useFooterDropDownList,
                                                options.showResizeHandle);
var returnValue; // Type: any
returnValue = instance.bandColumns(value);
    function bandColumns( 
   value : boolean
) : any;

    Parameters

    value
    Whether to display an alternating column style.

    Return Value

    If no value is set, returns whether to display an alternating column style; otherwise, returns the table.
